# Contributing

SpecDrift links PRD sections (Google Docs) to Jira issues and GitHub PRs to detect “spec drift” early.

## How to help

- Open an Issue with a clear problem statement, repro steps, and expected behavior.
- For PRs, keep changes small and include tests where possible.

## Project layout

- `specdrift/` — Python engine + CLI
- `dashboard/` — React dashboard (Vite)
- `.github/workflows/` — scheduled sync + PR checks

Thanks for improving SpecDrift.
